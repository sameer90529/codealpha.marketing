# Customer Segmentation and Wine Spending Prediction Using Machine Learning

## Overview

This project applies Machine Learning techniques to analyze customer purchasing behavior. It performs customer segmentation using K-Means Clustering and predicts wine spending using Linear Regression.

The objective is to identify customer groups based on income and spending patterns and estimate future spending for new customers.

---

## Features

* Data Cleaning and Preprocessing
* Customer Age Calculation
* Exploratory Data Analysis (EDA)
* Customer Segmentation using K-Means Clustering
* Wine Spending Prediction using Linear Regression
* Data Visualization with Matplotlib

---

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-Learn
* Google Colab

---

## Dataset Attributes

The project uses the following features:

| Feature    | Description                   |
| ---------- | ----------------------------- |
| Year_Birth | Customer birth year           |
| Income     | Annual income                 |
| MntWines   | Amount spent on wine products |
| Age        | Calculated customer age       |

---

## Project Workflow

### 1. Data Loading

Upload and load the dataset into Google Colab.

### 2. Data Cleaning

* Calculate Age from Year_Birth
* Select relevant features
* Remove missing values

### 3. Data Visualization

Generate visualizations:

* Income vs Wine Spending
* Age vs Wine Spending

### 4. Customer Segmentation

Apply K-Means Clustering to group customers based on:

* Income
* Wine Spending

### 5. Wine Spending Prediction

Train a Linear Regression model using:

* Age
* Income

Target Variable:

* MntWines

### 6. Prediction

Predict wine spending for new customers based on their age and income.

---

## Machine Learning Models

### K-Means Clustering

Used to segment customers into different groups based on purchasing behavior.

### Linear Regression

Used to predict wine spending using customer demographics.
